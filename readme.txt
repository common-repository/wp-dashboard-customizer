== Description ==

Customize your WordPress Dashboard easily. This plugin specially created for public blog/website.

= Features =

* Hide Screen Options Tab
* Hide Help Tab
* Remove Admin Bar ( Excerpt Admin )
* Restrict author to view all posts
* Restrict author to view all media
* Show Thumbnail in Dashboard
* Display Cut Copy Button in WordPress Editor
* Custom Login Logo
* Custom Message on Login Page
* Custom Message on Registration Page
* Remove WordPress Logo
* Remove About WordPress Link
* Remove WordPress.org Link
* Remove Documentation Link
* Remove Support Forums Link
* Remove Feedback Link
* Remove Right Now
* Remove Recent Comments
* Remove Recent Drafts
* Remove Plugins
* Remove Incoming Links
* Remove Quickpress
* Remove WordPress Blog
* Remove Other WordPress News

== Thanks ==
Thank you for using Wp Dashboard Customizer.

== Installation ==

1. Upload 'wp-dashboard-customizer' to the wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to settings => Wp DB Customizer and customize your settings.
